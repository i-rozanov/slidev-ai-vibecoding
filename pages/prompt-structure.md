# Детальный разбор структуры промпта

## Контекст: память диалога

::left::

**Что включает контекст:**
- История предыдущих сообщений
- Уже решенные задачи
- Установленные ограничения
- Предыдущие ошибки и исправления

**Пример:**
```python
# Контекст помогает модели "помнить"
user: "Создай функцию для валидации email"
assistant: "Вот функция на Python..."
user: "Теперь добавь проверку домена"
# Модель использует контекст предыдущего обмена
```

::right::

## Системный промпт: правила игры

**Определяет поведение модели:**
- Роль (разработчик, аналитик, etc.)
- Стиль ответов
- Ограничения и правила
- Формат вывода

**Пример системного промпта:**
```
Ты - опытный Python-разработчик. 
Отвечай кратко и по делу. 
Не предлагай небезопасные решения.
Проверяй код на ошибки перед отправкой.
```
